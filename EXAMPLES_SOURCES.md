# Examples Sourcing Manifest

Verified real-world example sources for every diagram, gathered via web research
(every URL was fetched live and the artifact visually confirmed by the research pass).

**Policy:** Mix — self-host openly-licensed images (CC/OGL/CC0/public-domain) with
attribution (Part A grid); link all real sources, incl. all-rights-reserved ones,
in Part B ("More examples to explore"). Fill each Part A grid to the 5-image target
with clearly-labeled **illustrative recreations** where open images fall short.

Legend: ✅ = self-hostable (open license + extractable image) · 🔗 = link-only
(ARR / gated / HTML-table-only / PDF-embedded).
